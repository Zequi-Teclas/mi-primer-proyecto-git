Descripción

Este proyecto es una aplicación web que desarrollé como parte de mi aprendizaje en desarrollo web. Incluye funcionalidades básicas como formularios, conexión a base de datos, y manejo de usuarios.

Objetivos

  • Aprender HTML, CSS y JavaScript
  • Implementar un sistema de usuarios
  • Conectar con una base de datos usando PHP

Tecnologías Utilizadas

Este proyecto fue desarrollado con:

  • HTML5
  • CSS3
  • JavaScript
  • PHP
  • MySQL
